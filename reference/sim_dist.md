# Simulation of single data block from distribution

Simulation of single data block from distribution

## Usage

``` r
sim_dist(num, n, p)
```

## Arguments

- num:

  Integer. Type of distribution. 1 for normal, 2 for uniform, 3 for
  exponential

- n:

  Integer. Number of data points.

- p:

  Integers. Number of variables in block.
